# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please send an e-mail to security@operation-paws.demo (Placeholder contact). 

We aim to address all security issues within 72 hours of disclosure.

## Scope
- Edge Functions and Database Policies
- Authentication and Role Management
- API integrations

Out of scope:
- Social engineering attacks
- Bugs in third-party libraries (unless actionable mitigation is possible locally)
