# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please **do not open a public issue**. Instead:

1. Use the **GitHub Private Vulnerability Reporting** feature.
2. Or email: `security-research@operation-paws.org.za` (Placeholder - monitored periodically).

We aim to acknowledge all security issues within 48 hours and provide a resolution plan within 72 hours.

## Scope
- Edge Functions and Database Policies
- Authentication and Role Management
- API integrations

Out of scope:
- Social engineering attacks
- Bugs in third-party libraries (unless actionable mitigation is possible locally)
