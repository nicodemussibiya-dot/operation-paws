# Operation PAWS — Roadmap (PAWS 1 → 500)

**Last updated:** 23 April 2026

This roadmap is designed to be measurable, auditable, and scalable.

## Phase A — Public foundation (Repo + QR hub)
Deliverables:
- GitHub Pages hub: `/docs/start/`
- Public tracker skeleton: `/docs/tracker/`
- PAWS reference spec + checksum

Exit criteria:
- QR hub is live and stable
- No-PII policy is enforced
- Tracker definitions are agreed

## Phase B — PAWS 1 → 10 (Pilot readiness)
Deliverables:
- pipeline SOPs and runbooks
- screening day checklist
- “not selected” humane outcomes policy
- verification page (`/docs/verify/`)

Exit criteria:
- stakeholder-ready pack exists (non-sensitive)
- clear action items list for official meeting(s)

## Phase C — PAWS 10 → 50 (Repeatable engine)
Deliverables:
- standard intake flow (WhatsApp/Telegram/web)
- Supabase public tracker connected (read-only)
- source stats (opt-in display)

Exit criteria:
- consistent throughput
- stable governance

## Phase D — PAWS Academy (Tier 3)
Deliverables:
- curriculum v1 (foundation only, non-tactical)
- Combine Day scorecard
- coach playbook
- safety + welfare SOP

Exit criteria:
- academy model runs off official premises
- objective scoring works

## Phase E — PAWS 100 → 500
Deliverables:
- multi-site operations kit
- sponsor programme
- optional compliant crypto rail (policy + audit)

Exit criteria:
- scale without compromising welfare, privacy, or security
