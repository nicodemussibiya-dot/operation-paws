# Operation PAWS — Privacy Notice (Repo & Website)

**Last updated:** 23 April 2026

Operation PAWS is a public-interest initiative. Privacy and safety matter.

This repository is public and does not store personal information. Any operational system that collects personal data must follow lawful processing principles (including purpose limitation and security safeguards) under applicable law (including POPIA where applicable).

## 1) What this public repo stores
This public GitHub repository may store:
- project documentation, templates (blank), policies, and code,
- public-safe IDs and aggregate statistics definitions.

It does **not** store:
- microchip numbers tied to people,
- owner/breeder/donor identities or contact details,
- signed documents,
- WhatsApp screenshots.

## 2) What the public website (GitHub Pages) may collect
If/when enabled, the Operation PAWS hub site may collect limited technical data such as:
- basic web analytics (page views, referrers) if configured,
- optional form submissions (only if you intentionally add forms).

If you add forms/chat, only collect the minimum needed, clearly state the purpose, and keep operational data ring-fenced.

## 3) Operational data (ring‑fenced system)
If/when an operational intake system is active (WhatsApp/Telegram/web intake + database), it may collect:
- contact details (name/number/email),
- dog details (age, breed, health notes),
- consent records,
- screening/assessment outcomes.

That operational data must be stored in a protected system with role-based access, audit logs, and retention rules.

## 4) Publishing rule
Operation PAWS will publish only:
- anonymised references (PAWS reference numbers),
- aggregate metrics,
- opt‑in public partner acknowledgements.

## 5) Data retention (recommended baseline)
- Leads not progressing: delete or anonymise after a defined period (e.g., 90–180 days), unless legal/operational reasons require retention.
- Accepted/assessed records: retain per partner governance requirements, then archive securely.

## 6) Contact
Do not open GitHub issues containing personal data.
